## dpokidov/imagemagick

I downloaded this container image on 2021-11-01.
Use docker-compose

### how to run

Use ```start.sh``` for the following case.

```
docker-compose up ; sleep 1 ; docker-compose run --rm magick
```
and get into interactive shell
